# Portfolio de Proyectos

Este proyecto es una colección de ejercicios y herramientas interactivas desarrolladas con HTML, CSS y JavaScript.

## Cómo ejecutar el proyecto

Para ver el portfolio y acceder a los ejercicios, simplemente abre el archivo `index.html` en tu navegador web favorito (Chrome, Firefox, Edge, etc.).

1. Navega a la carpeta del proyecto.
2. Haz doble clic en el archivo `index.html`.
3. Se abrirá el menú principal donde podrás seleccionar cualquiera de los proyectos disponibles.

## Proyectos Incluidos

- **Calculadora**: Operaciones matemáticas básicas.
- **Analizador de Texto**: Contador de palabras y caracteres.
- **Reloj Digital**: Muestra la hora actual.
- **Generador de Colores**: Crea paletas de colores aleatorias.
- **Intro al DOM**: Ejercicios de manipulación del DOM.
